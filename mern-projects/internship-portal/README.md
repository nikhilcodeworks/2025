# Git Internship Portal

A web-based internship portal where companies can post job/internship listings and students can explore and apply for opportunities. Designed to bridge the gap between companies seeking interns and students looking for hands-on experience.

## 🚀 Features

- ✅ Company registration and authentication
- ✅ Post internships/jobs with descriptions, requirements, and deadlines
- ✅ Student registration and login
- ✅ Explore available internships with filters and search
- ✅ Apply to internships with resume upload
- ✅ Admin panel to manage users and listings
- ✅ Email notifications for application status

## 💻 Tech Stack

**Frontend:**
- React.js
- TailwindCSS / Bootstrap
- Axios
- React Router DOM

**Backend:**
- Node.js
- Express.js
- MongoDB / Mongoose
- Multer (for file upload)
- JWT / OAuth2 (authentication)
- Nodemailer (email notifications)
